# Яндекс Погода Автокликер зонтиков ☂

Расширение для Chrome/Vivaldi, которое автоматически нажимает **«Нет»** на форме погодного отчёта Яндекс Погоды (карта «Зонтиков»).

## Возможности

- Автоклик по расписанию - настраиваемый интервал + случайная задержка
- Если кнопка не найдена - автоматический повтор через 3 минуты
- Переиспользует одну закреплённую вкладку - фокус крадётся только при первом создании; после клика вкладка сбрасывается на `about:blank`
- **Логирование** с тумблером вкл/выкл - persistent-логи в `chrome.storage.local`, просмотр/копирование/очистка из попапа
- **Кнопка «Проверить сейчас»** - ручной запуск клика для отладки
- **Тема оформления** - системная / тёмная / светлая (выбор сохраняется)
- **Тумблер «Логирование»** - выключает запись в консоль и storage
- Автостарт при запуске браузера
- Автооткрытие страницы, если вкладка закрыта
- Настройка интервала, задержки, URL через popup
- **Иконка** - зонтик на прозрачном фоне (16, 32, 48, 128 px)

## Установка

1. Открой `vivaldi://extensions` (или `chrome://extensions`)
2. Включи **Режим разработчика**
3. Нажми **«Загрузить распакованное расширение»**
4. Выбери папку с проектом

## Использование

1. Нажми на иконку расширения в панели
2. Включи тумблер **«Автоматизация»**
3. При необходимости включи **«Автооткрытие страницы»**
4. Расширение создаст закреплённую вкладку и будет кликать «Нет» по расписанию
5. **«Проверить сейчас»** - немедленный клик для проверки
6. **«Логи»** - просмотр / копирование / очистка журнала работы
7. **«Логирование»** - тумблер вкл/выкл записи логов
8. **Тема** - выбор между системной, тёмной и светлой

## Структура

```
├── icons/            # Иконки расширения (16, 32, 48, 128)
│   ├── icon16.png
│   ├── icon32.png
│   ├── icon48.png
│   └── icon128.png
├── manifest.json     # Манифест MV3
├── background.js     # Service Worker (таймер + управление вкладками + логи)
├── content.js        # Скрипт клика на странице
├── popup.html        # Интерфейс управления
├── popup.js          # Логика попапа
└── .gitignore
```

## Безопасность

Расширение работает в контексте вашего браузера с вашей сессией.
Не собирает данные, не отправляет запросы на сторонние серверы.
